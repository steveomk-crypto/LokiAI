# Dashboard Visual Style Spec

## Purpose
Define the visual identity of the rebuilt dashboard so it feels intentional, branded, and stream-worthy.

This spec is primarily for the **public stream dashboard**, though parts of the style system can be lightly adapted for the private operator dashboard.

## Creative Direction
The stream dashboard should feel like:
- **LokiAI operating a live market intelligence cockpit in space**
- a futuristic command bridge
- an AI pilot station monitoring signals, scans, systems, and intelligence outputs in real time

It should be cinematic, immersive, and high-energy without becoming cartoonish, scammy, or unreadable.

## Core Aesthetic
### Primary fantasy
**Hyper-real sci-fi cockpit / command bridge**

Visual references in spirit:
- spacecraft cockpit
- tactical HUD
- live operations console
- holographic financial intelligence station

### Emotional target
The dashboard should feel:
- alive
- intelligent
- powerful
- sharp
- premium
- exciting to watch over long periods

It should not feel:
- childish
- chaotic
- meme-casino cheap
- influencer scammy
- unreadable due to effects overload

## Brand Fit
This style must still match the LokiAI brand:
- sharp
- calm under pressure
- data-driven
- futuristic
- a little dramatic, but never desperate

The system should feel like a real operator machine in a cinematic setting.

## Visual Identity Themes
### Theme statement
**Neon intelligence in orbit.**

### Key ingredients
- deep space backdrop
- cockpit framing / viewport silhouette
- holographic panels
- layered glass/HUD effects
- neon telemetry highlights
- subtle ambient animation
- strong central focal area for live market intelligence

## Color System
### Primary colors
- **Electric cyan** — core data glow / live system energy
- **Neon violet / magenta** — accent energy and depth
- **Deep blue-black / void black** — background base
- **White / cool silver** — text and high-contrast highlights

### Secondary accent
- **Fox-gold / amber** for LokiAI identity touches
  - small use only
  - highlights, status accents, insignia, key CTAs

### Color behavior
- use neon sparingly for emphasis
- keep the base mostly dark and spacious
- preserve readability first
- avoid rainbow overload

## Background Direction
### Environment
The dashboard should sit inside a believable sci-fi environment.

Possible elements:
- stars / nebula / planetary glow outside cockpit glass
- faint atmospheric rim light
- dark metallic or holographic cockpit frame edges
- depth layers so the UI feels embedded in a real environment

### Important rule
Background should support the panels, not compete with them.
It must remain low enough contrast that text/data panels stay readable.

## Panel Style
### Panel design language
Panels should feel like:
- holographic consoles
- transparent smart-glass displays
- neon-edged data windows
- futuristic command modules
- side-console / cockpit control banks where appropriate

### Panel characteristics
- rounded but sharp-enough corners
- semi-transparent dark glass fill
- subtle neon borders
- inner glow on active states
- layered shadow depth
- occasional scanline / HUD micro-detail texture

### Avoid
- giant glowing boxes everywhere
- thick arcade outlines
- random gradients with no structure

## Typography
### Tone
Text should feel modern, technical, and premium.

### Heading style
- compact uppercase or tight futuristic styling
- crisp and authoritative
- not goofy “space font” nonsense

### Body text
- highly legible sans-serif
- should survive stream compression and mobile viewing

### Numeric displays
- strong monospaced or semi-monospaced option for prices, drifts, timestamps, and counts

## Motion / Animation Rules
### Desired motion
- subtle, continuous ambient motion
- breathing glows
- soft pulse on live status indicators
- occasional radar/sweep style movement
- gentle background drift/parallax
- animated data refresh transitions

### Hard rule
Motion should imply life, not distraction.
No constant flashy movement that makes the dashboard tiring to watch.

## LokiAI Presence
### Character presence
The dashboard should imply that **LokiAI is in the cockpit**.
This can be done with:
- silhouette / avatar frame
- AI operator badge
- “LokiAI Market Engine” branding
- system voice/status motifs
- fox-accent insignia or crest

### Important
Do not make it cheesy anime mascot mode unless explicitly desired later.
The presence should feel like:
- AI pilot
- systems operator
- command intelligence entity

Not a cartoon sticker slapped on a chart.

## Layout Atmosphere
### Focal hierarchy
The screen should read like a real command center:
- main intelligence zone in the center/left
- live pulse and scanner highlights most prominent
- support systems on flanks
- CTA/revenue zones integrated but not spammy

### Visual rhythm
- one strong hero area
- several support consoles
- one lower-motion footer/ticker area
- enough negative space to feel premium

## Stream-Specific Considerations
### Watchability
The stream dashboard should:
- look good from a distance
- still be readable when compressed by YouTube
- remain visually interesting during quiet markets
- feel like a persistent live environment

### Conversion without ugliness
Links/CTAs for:
- Substack
- Gumroad
- support/tips
- community
should be present but integrated tastefully into the design.

No screaming ad energy.

## Operator View Adaptation
The private operator dashboard can borrow the same system lightly:
- dark glass UI
- restrained neon accents
- cleaner layout
- less cinematic background
- much less visual drama

The operator dashboard should feel related, not identical.

## Visual Safety Rules
The aesthetic should never compromise:
- readability
- signal clarity
- trust
- professionalism

If an effect makes data harder to read, kill the effect.

## Implementation Guidance
### Build in layers
1. clean functional layout first
2. dark sci-fi shell second
3. neon/glass styling third
4. subtle motion fourth
5. visual polish after data reliability is proven

### Reusable style tokens to define later
- background colors
- panel glass colors
- border glow colors
- status colors
- typography sizes
- spacing scale
- blur level
- shadow/glow intensity
- animation speed presets

## Visual Goal Statement
When someone opens the stream, it should feel like:

**an AI operator is live in a spaceborne trading cockpit, monitoring the market in real time and turning raw motion into intelligence.**

That is the target.
